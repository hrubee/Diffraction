# `diffraction clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `diffraction clawbot qr` (same behavior as [`diffraction qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `diffraction clawbot qr` -> `diffraction qr`
